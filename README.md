**Project Overview**

This project focuses on analyzing customer churn data to identify key factors that influence customer attrition. The dashboard provides actionable insights to help businesses reduce churn and improve customer retention strategies.

The analysis was performed using Power BI with data cleaning, transformation, and visualization techniques

---

**Key Points of the Dashboard**

* Analyzed total customers and identified overall churn rate (%).
* Compared churned vs retained customers using interactive visuals.
* Identified higher churn among inactive members.
* Observed impact of credit score categories on customer attrition.
* Analyzed churn distribution by gender and geography.
* Compared churn based on number of products and account balance.
* Created DAX measures for KPIs like churn rate and customer count.
* Used interactive filters and slicers for dynamic analysis.
